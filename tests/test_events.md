# 이벤트 감지 테스트 체크리스트

이 문서는 RE_bHaptics 모드의 이벤트 감지 시스템을 수동으로 테스트하기 위한 체크리스트입니다.

## 테스트 준비

### 환경 설정
- [ ] REFramework 설치 완료
- [ ] 모드 스크립트 복사 완료
- [ ] 게임 정상 실행 확인
- [ ] `Insert` 키로 REFramework 메뉴 접근 가능

### 초기 확인
- [ ] ScriptRunner에서 에러 없음
- [ ] `RE_bHaptics` 항목이 Script-generated UI에 표시됨
- [ ] Debug Window 활성화 가능
- [ ] 콘솔/로그에 초기화 메시지 출력됨

---

## RE7 테스트

### 체력 감지

| 테스트 | 방법 | 예상 이벤트 | 결과 |
|--------|------|-------------|------|
| 피격 감지 | 적에게 피격당하기 | `player_hit` | ⬜ |
| 다중 피격 | 연속으로 피격당하기 | 각각 `player_hit` | ⬜ |
| 대량 피격 | 강력한 공격 받기 | `player_hit` (severity=critical) | ⬜ |
| 회복 감지 | 회복 아이템 사용 | `heal` | ⬜ |
| 심장박동 | 체력 30% 이하 유지 | `heartbeat` (주기적) | ⬜ |

### 피격 방향

| 테스트 | 방법 | 예상 방향 | 결과 |
|--------|------|-----------|------|
| 전방 피격 | 전방에서 공격받기 | `front` | ⬜ |
| 후방 피격 | 뒤에서 공격받기 | `back` | ⬜ |
| 좌측 피격 | 왼쪽에서 공격받기 | `left` | ⬜ |
| 우측 피격 | 오른쪽에서 공격받기 | `right` | ⬜ |

### 무기 이벤트

| 테스트 | 방법 | 예상 이벤트 | 결과 |
|--------|------|-------------|------|
| 권총 발사 | 권총 발사 | `gun_fire` | ⬜ |
| 샷건 발사 | 샷건 발사 | `gun_fire` | ⬜ |
| 재장전 | 무기 재장전 | `gun_reload` | ⬜ |
| 나이프 공격 | 나이프로 공격 | `melee_attack` | ⬜ |
| 연속 발사 | 빠르게 연사 | 각각 `gun_fire` | ⬜ |

## 문제 해결

### 이벤트가 감지되지 않을 때

1. REFramework 로그 확인
2. Object Explorer에서 해당 클래스 존재 확인
3. 메서드 후킹 성공 여부 로그 확인
4. 게임 버전과 설정 파일 호환성 확인
